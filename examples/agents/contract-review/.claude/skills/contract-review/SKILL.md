# contract-review 스킬

계약서 조항을 분석하는 4단계 orchestrator-workers 워크플로우.

## 절차
1. `clause-extractor`로 계약서 전체에서 조항 번호·제목·내용 구조 목록을 추출한다.
2. `risk-analyst`로 각 조항의 리스크 유형·심각도·근거를 조항마다 별도로 분석한다 (N회 반복 호출).
3. `cross-checker`로 조항 간 충돌 쌍과 누락된 보호 조항을 교차검증한다.
4. `summary-writer`로 Top 3 리스크, 협상 포인트, 전체 위험도 총평을 종합 작성한다.

## 출력 형식
`## 핵심 조항` / `## 리스크 분석` / `## 교차검증` / `## 요약 의견` 네 섹션의 마크다운.
최종 출력에는 "이 분석은 법률 자문이 아닌 참고용입니다." 문구가 반드시 포함된다.
