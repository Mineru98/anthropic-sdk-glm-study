# copy-optimize 스킬

광고 카피를 evaluator-optimizer 패턴으로 개선하는 3단계 워크플로우.

## 절차
1. `writer`로 제품/타깃/채널 정보를 바탕으로 카피 초안(번호 목록)을 작성한다.
2. `critic`으로 초안 전체를 평가하고 설득력·브랜드적합성 기준으로 피드백과 개선점을 받는다.
3. `finalizer`로 초안과 피드백을 모두 반영한 최종 카피를 확정한다. 각 카피에 "(개선: …)" 주석을 병기한다.

## 출력 형식
`## 초안` / `## 피드백` / `## 최종 카피` 세 섹션의 마크다운.
