# finance-analysis 스킬

기업 재무 데이터를 5단계 orchestrator-workers 패턴으로 분석하는 헤비 워크플로우.

## 절차

1. `data-extractor`로 자연어 재무 텍스트에서 핵심 수치를 정형화된 목록으로 추출한다.
2. `metric-analyst`에 추출 결과를 넘겨 성장성/수익성/안정성 3개 축 지표를 분석한다.
3. `risk-assessor`에 수치와 분석 결과를 함께 전달해 재무·시장 리스크를 항목별로 평가한다.
4. `number-verifier`에 1~3단계 결과 전체를 전달해 계산 비율 및 수치 일관성을 검증한다.
5. `opinion-writer`에 1~4단계 결과 전체를 전달해 투자 의견(매수/중립/매도/보유/주의)을 근거와 함께 작성한다.

## 출력 형식

`## 핵심 지표` / `## 분석` / `## 리스크` / `## 투자 의견` 네 섹션의 마크다운.
`## 투자 의견` 섹션 끝에는 반드시 "본 리포트는 투자 권유가 아닌 참고용입니다." 면책 문구가 포함되어야 한다.
