# triage 스킬

고객 문의를 3단계 routing 워크플로우로 처리해 담당 팀까지 배정한다.

## 절차
1. `classifier`로 카테고리(결제/배송/계정/기술/환불/기타) 분류.
2. `priority-scorer`로 P1~P4 긴급도 산정 (원문 + 카테고리 전달).
3. `router`로 담당 팀과 다음 조치 결정 (원문 + 카테고리 + 우선순위 전달).

## 출력 형식
`## 분류` / `## 우선순위` / `## 라우팅` 세 섹션의 마크다운.
