# meeting-pipeline 스킬

회의록을 체계적으로 분석하는 3단계 prompt-chaining 워크플로우.

## 절차
1. `summarizer`로 회의록 원문의 핵심 내용을 3–5문장으로 요약한다.
2. `decision-logger`로 원문과 요약을 바탕으로 확정 결정사항(보류 포함)을 목록화한다.
3. `action-extractor`로 원문·요약·결정사항을 모두 참고해 담당자와 기한이 명시된 액션 아이템을 추출한다.

## 출력 형식
`## 요약` / `## 결정사항` / `## 액션 아이템` 세 섹션의 마크다운.

## 체이닝 규칙
각 단계는 이전 단계의 결과를 누적 전달받아 처리한다. action-extractor는 원문·요약·결정사항 세 가지를 모두 받아야 가장 정확한 액션 아이템을 산출한다.
