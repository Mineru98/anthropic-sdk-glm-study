# ecommerce-listing 스킬

원시 상품 정보를 받아 판매 준비 완료된 제목·설명·SEO 태그를 생성하는 3단계 prompt-chaining 워크플로우.

## 절차
1. `attribute-extractor`로 원시 상품 정보에서 제품명·핵심 기능·스펙·가격 등 판매에 중요한 속성을 "속성: 값" 목록으로 추출한다.
2. `copywriter`로 속성 목록을 바탕으로 클릭을 유도하는 제목(30자 이내)과 3~5줄의 상품 설명을 작성한다.
3. `seo-tagger`로 속성 목록과 카피 결과를 바탕으로 소비자 검색 의도에 맞는 #해시태그를 5개 이상 생성한다.

## 출력 형식
`## 제목` / `## 상품 설명` / `## SEO 태그` 세 섹션의 마크다운.
